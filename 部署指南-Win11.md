## 📋 部署清单

### 第一步：下载微信开发者工具

**下载地址:** https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

**选择版本:**
- ✅ Windows 64 位
- ✅ 稳定版 (Stable Build)

**安装步骤:**
1. 下载安装包 (约 200MB)
2. 双击运行安装程序
3. 选择安装路径 (建议默认 `C:\Program Files (x86)\Tencent\WeChatDevTools`)
4. 完成安装

---

### 第二步：获取项目代码

#### 方式 A: Git 克隆 (推荐)

```bash
# 打开 PowerShell 或 CMD
cd C:\Users\你的用户名\Documents\WeChatProjects
git clone https://github.com/oshimadai/wechat-link-game.git
cd wechat-link-game
```

#### 方式 B: 下载 ZIP

1. 访问 https://github.com/oshimadai/wechat-link-game
2. 点击绿色 "Code" 按钮
3. 选择 "Download ZIP"
4. 解压到 `C:\Users\你的用户名\Documents\WeChatProjects\wechat-link-game`

---

### 第三步：导入项目到开发者工具

1. **打开微信开发者工具**
   - 桌面快捷方式或开始菜单

2. **扫码登录**
   - 使用你的微信扫码登录
   - 需要是小程序管理员或开发者

3. **导入项目**
   - 点击 "+" 或 "导入项目"
   - **项目路径:** 选择 `wechat-link-game` 文件夹
   - **AppID:** 
     - 如果有小程序账号 → 填写 AppID
     - 如果没有 → 选择 "测试号" 或 "小程序测试号"
   - 点击 "导入"

---

### 第四步：编译运行

1. **自动编译**
   - 导入后会自动编译
   - 左侧模拟器显示游戏画面

2. **手动编译**
   - 点击工具栏 "编译" 按钮 (▶️)
   - 或按 `Ctrl + R`

3. **查看效果**
   - 上下布局 (玩家 1 在上，玩家 2 在下)
   - 每名玩家 5 张手牌
   - 血量格子显示

---

### 第五步：真机测试

1. **点击"预览"**
   - 工具栏点击 "预览" 按钮
   - 或按 `Ctrl + Shift + B`

2. **扫码测试**
   - 使用微信扫描二维码
   - 游戏会在手机上打开

3. **双人测试**
   - 两人共用一台手机
   - 或分别扫码 (需要同一个测试号)

## 🔧 常见问题解决

### Q1: 编译报错 "app.json 找不到"

**原因:** 项目路径不正确

**解决:**
```bash
# 确认 app.json 在项目根目录
dir C:\Users\你的用户名\Documents\WeChatProjects\wechat-link-game\app.json
```

如果文件不存在，需要创建 `app.json`:

```json
{
  "pages": ["pages/index/index"],
  "window": {
    "backgroundTextStyle": "dark",
    "navigationBarBackgroundColor": "#2C3E50",
    "navigationBarTitleText": "🔗 1V1 连线桌游",
    "navigationBarTextStyle": "white"
  },
  "style": "v2",
  "sitemapLocation": "sitemap.json"
}
```

---

### Q2: 真机预览黑屏

**解决步骤:**

1. **检查 AppID**
   - 确认 `project.config.json` 中 AppID 正确

2. **清除缓存**
   - 工具 → 清除缓存 → 清除全部缓存

3. **重启工具**
   - 关闭微信开发者工具
   - 重新打开并编译

4. **检查网络**
   - 确保电脑和手机在同一 WiFi

---

### Q3: 卡牌点击没反应

**调试方法:**

1. **打开控制台**
   - 点击工具栏 "调试器" 标签
   - 查看 Console 中的错误信息

2. **检查回合**
   - 确认是当前玩家的回合
   - 查看中间显示 "▶ 玩家 X ◀"

3. **检查卡牌状态**
   - 卡牌是否已被触发 (变暗)
   - 是否正在等待能力目标

---

### Q4: 如何开始新游戏

**方法:**
1. 点击工具栏 "编译" 重新加载
2. 或在控制台输入 `initGame()`
