## 🎯 核心规则（已实现）

### 手牌系统
- ✅ 每名玩家固定 **5 张手牌**
- ✅ 牌组 70 张（火 22 + 水 22 + 木 22 + 万能 4）
- ✅ 触发后补充至 5 张

### 连线规则
- ✅ 只能连接**相同属性**的卡牌
- ✅ 万能卡可以连接任意属性
- ✅ 箭头方向决定连线路径（↑↓→←）
- ✅ 连线长度≥3 张可获得 Token

### 衰减规则
- ✅ 每回合结束，**未触发的卡牌 -1 血**
- ✅ 血量归零时卡牌死亡，扣 1 分
- ✅ ∞血卡不会衰减

### 胜利条件
- ✅ 率先达到 **20 分** 的玩家获胜
- ✅ 对方获得公平回合

## 🚀 使用方法

### 在微信开发者工具中打开

1. 打开微信开发者工具
2. 导入项目：选择 `wechat-link-game` 文件夹
3. 填写自己的 AppID（或使用测试号）
4. 编译运行

### 真机预览

1. 点击"预览"按钮
2. 使用微信扫描二维码
3. 在手机上测试游戏
