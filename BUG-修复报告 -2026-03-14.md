## 🐛 修复的 Bug

### BUG-002 (P0 紧急) - ✅ 已修复
**问题:** 点击激活后，应该使用功能，使用后开始按照箭头方向自动连接消除；消除后应该增加对应分数

**原因:**
- `activateSelectedCard()` 函数只调用了 `triggerCard()` 但没有正确处理后续流程
- `checkChain()` 函数在卡牌已触发后无法找到原始卡牌信息

**修复:**
1. 重写 `activateSelectedCard()` - 先触发卡牌，再检查连线，最后重置选择
2. 修改 `checkChain()` - 使用 `gameState.selectedCard` 保存的原始卡牌信息进行连线判断
3. 确保计分逻辑在 `triggerCard()` 中正确执行

**代码变更:**
```javascript
// activateSelectedCard - 修复后
function activateSelectedCard() {
  if (!gameState.selectedCard) return
  const { player, index, card } = gameState.selectedCard
  
  // 先触发卡牌（计分并从手牌移除）
  triggerCard(player, index)
  
  // 然后检查连线
  checkChain(player, index)
  
  // 重置选择
  gameState.selectedCard = null
  render()
}
```
