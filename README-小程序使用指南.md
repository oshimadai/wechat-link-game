## 🚀 在微信开发者工具中运行

### 步骤 1: 打开微信开发者工具

下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

1. 点击 "+" 或 "导入项目"
2. 项目路径：选择 `wechat-link-game` 文件夹
3. AppID：
   - 如果有：填写你的小程序 AppID
   - 如果没有：选择 "测试号"
4. 点击 "导入"

1. 点击工具栏的 "编译" 按钮
2. 在模拟器中查看效果

### 步骤 4: 真机测试

1. 点击 "预览"
2. 使用微信扫描二维码
3. 在手机上体验游戏

## 🎮 游戏玩法

### 基本规则

1. **目标**: 率先达到 20 分
2. **手牌**: 每名玩家 5 张
3. **连线**: 相同属性才能连接
4. **衰减**: 未触发的卡牌每回合 -1 血
5. **胜利**: 先达到 20 分获胜

### 操作流程

1. **选择卡牌**: 点击自己区域的牌
2. **使用能力**（如果有）:
   - 🔄 旋转：改变箭头方向
   - 🎨 变色：改变属性
   - 💫 换位：交换两张牌位置
3. **触发连线**: 自动计算分数
4. **衰减阶段**: 未触发卡牌 -1 血
5. **补充手牌**: 补到 5 张
6. **交换回合**

### 卡牌属性

- 🔥 **火**: 红色血量格子
- 💧 **水**: 蓝色血量格子
- 🌿 **木**: 绿色血量格子
- 🌈 **万能**: 紫色血量格子，可连任意属性

### 血量与分数

| 血量 | 分数 | 说明 |
|------|------|------|
| 3 血 | 1 分 | 常见 |
| 2 血 | 2 分 | 中等 |
| 1 血 | 3 分 | 高分但易死 |
| 0 血 | 0 分 | 立即死亡 |
| ∞血 | 1 分 | 不会衰减 |

## 🐛 常见问题

### Q1: 编译报错 "app.json 找不到"
**解决**: 确认 `app.json` 在项目根目录

### Q2: 真机预览黑屏
**解决**: 
1. 检查 AppID 是否正确
2. 清除缓存重新编译
3. 重启微信开发者工具

### Q3: 卡牌点击没反应
**解决**: 
1. 检查是否在正确的回合
2. 确认卡牌未被触发过
3. 查看控制台是否有错误

### Q4: 如何重置游戏
**解决**: 点击 "开始新游戏" 按钮
